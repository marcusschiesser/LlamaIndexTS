title: Ollama
---

To use Ollama embeddings, you need to import `OllamaEmbedding` from `@vectorstores/ollama`.

Note that you need to pull the embedding model first before using it.

In the example below, we're using the [`nomic-embed-text`](https://ollama.com/library/nomic-embed-text) model, so you have to call:

```shell
ollama pull nomic-embed-text
```

## Installation

```package-install
npm i llamaindex @vectorstores/ollama
```

```ts
import { OllamaEmbedding } from "@vectorstores/ollama";
import { Document, Settings, VectorStoreIndex } from "@vectorstores/core";

Settings.embedModel = new OllamaEmbedding({ model: "nomic-embed-text" });

const document = new Document({ text: essay, id_: "essay" });

const index = await VectorStoreIndex.fromDocuments([document]);

const queryEngine = index.asQueryEngine();

const query = "What is the meaning of life?";

const results = await queryEngine.query({
  query,
});
```
