---
title: Embedding
---

The embedding model in LlamaIndex is responsible for creating numerical representations of text. By default, LlamaIndex will use the `text-embedding-ada-002` model from OpenAI.

This can be explicitly updated through `Settings.embedModel`.

## Installation

```package-install
npm i llamaindex @vectorstores/openai
```

```typescript
import { OpenAIEmbedding } from "@vectorstores/openai";
import { Settings } from "@vectorstores/core";

Settings.embedModel = new OpenAIEmbedding({
  model: "text-embedding-ada-002",
});
```

## Local Embedding

For local embeddings, you can use the [HuggingFace](/framework/modules/models/embeddings/huggingface) embedding model.
