title: VoyageAI
---

To use VoyageAI embeddings, you need to import `VoyageAIEmbedding` from `@vectorstores/voyage-ai`.

## Installation

```package-install
npm i llamaindex @vectorstores/voyage-ai
```

```ts
import { VoyageAIEmbedding } from "@vectorstores/voyage-ai";
import { Document, Settings, VectorStoreIndex } from "@vectorstores/core";

Settings.embedModel = new VoyageAIEmbedding();

const document = new Document({ text: essay, id_: "essay" });

const index = await VectorStoreIndex.fromDocuments([document]);

const queryEngine = index.asQueryEngine();

const query = "What is the meaning of life?";

const results = await queryEngine.query({
  query,
});
```
